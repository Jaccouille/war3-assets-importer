Importateur de modèles Warcraft 3

Importe des assets personnalisés (modèles 3D, textures, sons et tout autre fichier) dans un fichier de carte Warcraft III (.w3x / .w3m), et crée optionnellement des définitions d’unités pour que les modèles soient utilisables dans le World Editor et en jeu.

Démarrage rapide

  1. Cliquez sur Ouvrir la carte pour charger une carte .w3x ou .w3m.
  2. Cliquez sur Importer le dossier de modèles pour analyser un dossier contenant les assets à importer.
  3. Sélectionnez les fichiers à importer à l’aide des cases à cocher dans l’onglet Ressources.
  4. Configurez les options dans l’onglet Configuration d’import.
  5. Cliquez sur Traiter et enregistrer — la carte traitée est écrite dans le chemin de sortie.

Onglet Ressources

Affiche les assets découverts selon deux vues sélectionnables :

  • Vue par catégorie — fichiers regroupés par type (Modèles / Textures), organisés dans la même hiérarchie de dossiers que sur le disque.
  • Vue par dossier — l’arborescence brute du système de fichiers à partir du dossier sélectionné. Affiche le nombre de fichiers/dossiers et la taille totale entre parenthèses.

Utilisez les cases à cocher pour sélectionner les fichiers à importer. Cocher un nœud de dossier sélectionne tout son contenu. Cliquer sur un nœud de fichier affiche un aperçu de l’asset à droite.

Onglet Configuration d’import

Création d’unités

Créer des unités Ajoute une définition d’unité personnalisée pour chaque modèle MDX sélectionné. Chaque unité apparaît comme une nouvelle entrée dans l’Éditeur d’objets (fichier W3U).
Placer les unités Place les unités créées sur le terrain de la carte. Dessinez un rectangle sur l’aperçu de la carte pour définir la zone de placement, ou laissez-le vide pour utiliser toute la zone délimitée par la caméra.
Supprimer toutes les unités Supprime tous les placements d’unités existants de la carte avant l’import. Utile pour recommencer depuis zéro sans accumuler les doublons.
Supprimer les assets existants Supprime tous les assets précédemment importés de l’archive de la carte avant d’ajouter la nouvelle sélection. Utile pour remplacer un import sans laisser de fichiers obsolètes.
Garder le nom de fichier Stocke les assets importés sans leur chemin de sous-dossier (ex. Infantry/soldier.mdx → soldier.mdx). Les textures de bouton (BTN*/DISBTN*) utilisent toujours leur chemin WC3 canonique quelle que soit cette option.
Échelle de l’unité Facteur d’échelle appliqué à la taille du modèle des unités créées. 1.0 = taille originale, 2.0 = double taille.

Placement des unités

ID d’unité d’origine ID du type d’unité de base utilisé comme modèle pour les unités créées (ex. hfoo pour le Fantassin humain). Les nouvelles unités héritent de leurs statistiques de base de ce type. Saisissez du texte pour rechercher les définitions d’unités existantes dans la carte chargée.
Angle de l’unité Direction de face des unités placées en degrés. 0° = Est   90° = Nord   180° = Ouest   270° = Sud (direction caméra WC3 par défaut).
Espacement X / Y Écart horizontal et vertical entre les unités sur la grille de placement, mesuré en pixels par rapport à l’image de carte affichée. Des valeurs plus petites regroupent les unités plus étroitement.
Ordre de placement Indique si la grille de placement se remplit de gauche à droite ligne par ligne, ou de haut en bas colonne par colonne.

Nommage des unités

Nommer depuis le fichier MDX Définit automatiquement le nom d’affichage de chaque unité en convertissant son nom de fichier MDX. Choisissez un style avec la liste déroulante Format.
Assigner l’icône BTN Si une texture BTN* correspondant au nom du fichier MDX est trouvée parmi les assets sélectionnés, elle est automatiquement assignée comme icône de la carte de commandes de l’unité.
Format Comment le nom de fichier est converti en nom d’affichage :
  Séparé par espaces (casse conservée) : InfantryAmerican → Infantry American
  Séparé par espaces : → Infantry american
  camelCase : → infantryAmerican
  snake_case : → infantry_american
  UPPER_CASE : → INFANTRY_AMERICAN
Aperçu Ouvre un tableau montrant comment chaque nom de fichier MDX sélectionné sera formaté avec les paramètres actuels.

Aperçu de la carte

Le panneau de droite affiche l’image de la carte et permet de définir la zone de placement des unités.

  • Cliquez et faites glisser pour dessiner un rectangle sur l’image de la carte.
  • Les triangles bleus indiquent où les unités seront placées avec l’espacement actuel.
  • Les triangles jaunes représentent les unités déjà présentes dans la carte.
  • Cliquez sur Effacer pour supprimer la forme dessinée ; les unités utiliseront alors toute la zone de la carte.
  • L’indicateur d’état est vert lorsque toutes les unités tiennent dans la zone, rouge lorsque la zone est trop petite (agrandissez la forme ou augmentez les valeurs d’espacement).
